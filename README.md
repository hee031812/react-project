# react를 이용한 포트폴리오 사이트 만들기

## 작업순서

1. 리액트 설치
2. git에 업로드
3. [lenis 사이트] (https://github.com/studio-freight/lenis)

## 설치

1. react 설치 'npx create-react-app 프로젝트 이름 '
2. gsap 설치 'npm i gsap'
3. sass 설치 'npm i sass'
4. lenis 설치 'npm i @studio-freight/lenis'
